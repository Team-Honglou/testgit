Update Plugin

Type the following on the command line to update {{plugin.name}}.

logdisplayplatform-cli plugins update {{plugin.id}}
Check out {{plugin.name}} on LogDisplayPlatform.com for README and changelog. If you do not have access to the command line, ask your LogDisplayPlatform administator.

Pro tip: To update all plugins at once, type logdisplayplatform-cli plugins update-all on the command line.
